This is a Julia project for quantum graph homology.

## Installation
```shell
> git clone https://github.com/XingyuZhang2018/qgh
```
move to the file and run `julia REPL`, press `]` into `Pkg REPL`
```julia
(@v1.10) pkg> activate .
Activating project at `~/../qgh`
```

```julia
(qgh) pkg> instantiate
```
